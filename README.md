### 利用pygame進行的遊戲開發學習<br>
* **遊戲介紹**
  玩家可透過鍵盤a和d對遊戲中機體進行左右移動，
  並利用空格鍵進行子彈的發射。
  吃到閃電可以提升機體，吃到能量可以回血。
  每10秒鐘會出一次BOSS，BOSS在血量為50%及25%時會提升能力，增加攻擊速度。
  持續挑戰存活可以獲得分數!
